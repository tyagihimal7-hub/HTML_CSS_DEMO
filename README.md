# ShopFast – A Shopping Website (HTML + CSS)
This is a frontend-only shopping website built using HTML and CSS.
The goal was to understand layout structuring, Flexbox, responsive components, and UI design fundamentals.

Technologies Used :- 
1. HTML5
2. CSS3
3. Flexbox
4. Font Awesome Icons

Features :-
1. Sticky navigation bar
2. Search bar UI
3. Category grid section
4. Hero banner
5. Footer with multiple sections
6. Hover effects and minimal animations
7. Clean folder structure

Folder Structure :-
    ShopFast/
    │── index.html
    │── style.css
    │── /images
    │── README.md

Future Improvements :-
1. Add JavaScript for dynamic search
2. Add login/signup page
3. Make it fully responsive for all screen sizes
4. Add cart functionality
5. Integrate backend later
